// WHAT IS $LIBERTY?

$LIBERTY is a decentralized AI coordination token.

It replaces centralized control with a trust layer — your token balance determines how much influence you have over AI agents in the swarm.

// WHAT IS AN AI SWARM?

A swarm is a network of AI agents working in parallel — each specializing in a task. They coordinate without a central controller.

Think of it as a hive: no queen gives orders, yet the system solves complex problems together.

// HOW DOES TRUST WORK?

Each agent reads your wallet's $LIBERTY balance before processing your request.

More tokens = higher trust tier = priority routing, richer outputs, and faster responses.

Low-trust requests are queued or rate-limited. Trust is earned, not granted.
